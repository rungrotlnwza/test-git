1. index.html
   เป็นหน้าแรกของเว็ปไซต์ ใช้สำหรับแสดงหน้าเริ่มต้นให้ผู้ใช้งานในหน้านี้ ไม่มีการเรียกใช้ api
   โดยส่วนใหญ่จะใช้สำหรับ

- แสดงข้อมูลทั่วไป
- ปุ่มไปหน้า login หรือ register

2. Login.html
   เป็นหน้าสำหรับให้ผู้ใช้งานเข้าสู้ระบบ

ข้อมูลที่ต้องส่งไปยัง api

- username
- password

  การทำงาน
  เมื่อผู้ใช้กรอกข้อมูลและกด login
  ระบบจะทำการส่งข้อมูลไปยัง api เช่น /api/login
  โดยใช้ method post เพื่อตวรจสอบความถูกต้องของข้อมูล

3. register.html
   เป็นหน้าสำหรับสมัครสมาชิกใหม่

- username
- Email
- password
- confrime password

การทำงาน
เมื่อผู้ใช้กรอกข้อมูลครบและกดสมัคร
ระบบจะ

- ตรวจสอบว่า password และ confirm password ตรงกัน
- ส่งข้อมูลไปยัง api /api/register ด้วย method post

ผลลัพธ์

- ถ้าสมัครสำเร็จ ไปหน้า login.html
- ถ้าไม่สำเร็จ แสดง error เช่น email ซํ้า

4. dashboard.html
   เป็นหน้าหลักหลังจากผู้ใช้เข้าสู่ระบบสำเร็จ

ข้อมูลที่ต้องรับมา

- firstname
- LastName
- Email

การทำงาน

- ดึงข้อมูลผู้ใช้จาก api เช่น /api/user
- แสดงข้อมูลลบหน้า dashbord
